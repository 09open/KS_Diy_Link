# Jlink_V9_DIY
 个人DIY JLINK V9  高速SWD速度可达12000kHz
 串口波特率最高支持230400
 DFU下载固件时需要将BOOT0短接3.3  RX引脚短接GND即可进入DFU模式
 进入DFU模式后刷入bootloader后打开jlink commander即可自动升级固件

 Exec SetSN=20381318    V9.3
